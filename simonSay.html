🎮 Simon Says

Press any key to start the game

📜 Game Rules

  • Press any key to start the game.
  • Watch the color that flashes.
  • Click the colors in the same order.
  • Each level adds one new color.
  • Repeat the complete sequence correctly.
  • If you click the wrong color, the game ends.
  • Try to reach the highest level!
💡 Tip:
Memorize the pattern instead of individual colors. The sequence gets longer every level.
